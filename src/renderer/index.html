📖 Instructions & Examples
Learn how to configure FZautochoice to automate dialog buttons

Basic Usage Flow

1. **Configure Match Text**: Enter a portion of text from the target dialog title or body in the **Detection** tab.
2. **Add Options**: Add potential buttons to detect (e.g. Yes, No, Allow, OK) and check the checkbox to make them Auto-Click.
3. **Set Action Targets**: In the **Actions** tab, define which text targets should receive the simulated clicks.
4. **Start Scan**: Click the ▶ Start Monitoring button or press the global hotkey `Ctrl+Shift+F9`.
